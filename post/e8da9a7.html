Hexo+Butterfly+Valine+Github搭建博客

发表于2023-03-08|更新于2023-03-09
|字数总计:1.3k|阅读时长:4分钟|阅读量:|评论数:

搞了这么多天，现在已经基本拥有个人博客所需要具备的全部功能了，另外还添加了一点个人的创意。这么一段时间，遇到的bug不计其数, 分享一下我制作BLOG时的相应流程与具体代码实现，认真看完这篇博客，你做不出来你直接喷我！

当前BLOG拥有的功能有：

  • 首页浮动体循环每日一句

  • 首页自定义背景

  • 自定义头像与web icon

  • 自定义侧边栏

  • 博客版权声明

  • 打赏功能（懂我意思吧）

  • 博客下方评论区发表评论，支持emoji

  • 白天/黑夜模式切换

  • 鼠标点击与打字特效（需要PC端实现）

  • 板娘（dddd）(需要科学上网方可流畅互动)

下面我将逐一说明上述功能的代码与需要注意的点（快进到安装好node.js，git，使用hexo框架并且连接GitHub仓库后，这里不再详细叙述，如有不懂的可以评论区留言或者去B站学习）

每日一句 / 网站名 主题配置文件添加，下同：

自定义背景 / Web Icon / 头像

这里需要注意的是，所使用的图片最好在本地进行下载以后再发布，可以显著降低时延

自定义侧边栏

Social里可以提供你的任何联系方式，包括但不限于Email、QQ、Wechat、Github、CSDN

！打赏功能 ！

这里可以添加wx支付和其他支付方式，实现方法与Alipay一样，付款码需下载到本地再进行上传

Valine 评论区

这里使用Valine需要调用LeanCloud平台，上述的API key和API id都是LC平台提供，如何登陆并使用LC平台在此不表，自行查阅

鼠标点击与打字特效（only for PC）

板娘 (需要科学上网，否则将可能出现板娘消失的情况)

经过张书樵大神魔改后的项目下载地址：https://github.com/stevenjoezhang/live2d-widget

下载解压到：themes\butterfly\source\文件夹下

打开项目目录进入修改autoload.js文件，将live2d_path设为自己的路径，一般没什么太大变化都为：/live2d-widget/

打开butterfly下的layout下的includes目录，点击修改head.pug文件，在文件末尾粘贴：

最后在主题配置文件下即butterfly下的_config.yml中开启看板娘：
